# 🌱 Plantify AI

**Plantify AI** is an IoT and AI-based smart agriculture system designed to monitor plant health in real time and detect plant diseases using image processing and machine learning.

The system helps farmers and researchers observe environmental conditions, identify plant types, detect diseases, and receive treatment suggestions.

---

# 🚀 Project Overview

Plantify AI integrates **ESP32 microcontrollers, environmental sensors, and AI image classification** to create an intelligent plant monitoring system.

The system performs the following tasks:

- Collects environmental data such as **temperature, humidity, and soil moisture**
- Captures **plant images using ESP32-CAM**
- Uses **AI models to identify plants and detect diseases**
- Provides **treatment and prevention recommendations**
- Stores and displays information through a **web-based interface**

This project aims to support **smart farming and precision agriculture** by enabling early disease detection and better crop management.

# 🔍 Features

- 🌡 **Real-time Temperature Monitoring**

### Hardware
- ESP32
- ESP32-CAM
- DHT11 Temperature & Humidity Sensors
- Soil Moisture Sensor

- Arduino IDE
- Python (AI Model Development)

# 📌 ESP32 Pin Mapping

| Component | ESP32 Pin | Description |
|----------|-----------|-------------|
| DHT11 Sensor 1 | GPIO 13 | Temperature & humidity monitoring |
| DHT11 Sensor 2 | GPIO 14 | Secondary environmental sensor |
| Soil Moisture Sensor | GPIO 32 | Analog input for soil moisture |
| Servo Motor 1 | GPIO 4 | Camera rotation control |
| Servo Motor 2 | GPIO 5 | Sensor positioning |
| Motor IN1 | GPIO 26 | L298N motor driver |
| Motor IN2 | GPIO 25 | L298N motor driver |
| Motor IN3 | GPIO 33 | L298N motor driver |
| Motor IN4 | GPIO 27 | L298N motor driver |
